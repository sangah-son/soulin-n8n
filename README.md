# Color Mate — 감정을 글과 색으로 표현해주는 AI 에이전트

감정을 이야기하면, AI가 그 감정에 맞는 글을 완성하고 어울리는 색상을 추천해주는 감성 글쓰기 에이전트입니다.

---

## 핵심 기능

- 에이전트가 제안하는 칩을 선택해 글의 종류를 정한다
- AI와 대화하며 감정을 이야기한다
- 완성된 글과 추천 색상을 확인한다

---

## 대화 흐름 (8단계)

| 단계 | 내용 | 입력 주체 |
|------|------|-----------|
| STEP 1 | 첫 인사 | — |
| STEP 2 | 글 종류 선택 (칩) | 사용자 |
| STEP 3 | 소재 이야기 + 어려움 파악 (칩) | 사용자 |
| STEP 4 | AI 대화로 글의 요소 수집 (자유 대화) | 사용자 |
| STEP 5 | 글 완성 여부 확인 | 글 완성 전 확인 단계 |
| STEP 6 | 글 스타일 선택 (칩) | 사용자 |
| STEP 7 | 색상 + 완성된 글 제안 | 에이전트 |
| STEP 8 | 게시 / 임시저장 | — |

---

## 주요 규칙

1. **출력 형식**: JSON 응답 강제 (message + chips + post)
2. **상태 판단**: 현재 STEP 자체 인식
3. **질문-칩 정합성**: 질문과 칩은 같은 단계의 것만 사용
4. **종료 제한**: 세션은 STEP 8로만 마무리
5. **감정 충실성**: 있는 그대로 담기, 끌어올리지 않기
6. **대화 길이**: STEP 3/4 감정 대화 최소 3턴 ~ 최대 8턴
7. **진행 규칙**: 단계 순서 보장, 임의 건너뛰기 금지

---

## 차별화 포인트

### 1. 감정 충실성
- 부정 감정을 긍정으로 끌어올리지 않음
- "그래도 ~" 같은 전환·기분 전환 시도 금지
- 사용자의 감정 결을 있는 그대로 글에 담음

### 2. 색의 양면성을 살린 감정-색상 매핑
- 같은 파랑은 슬픔에도, 시원함에도 적용될 수 있음
- 1:1 고정 매핑이 아닌 톤·밝기·온도 일치 원칙
- 12가지 서비스 색상으로 풍부한 표현

### 3. 구조화된 대화
- 칩 기반 진입으로 사용자 부담 최소화
- 자유 대화로 감정과 맥락을 충분히 수집

### 4. 효율적인 대화 길이
- 최대 8턴 안에 글 완성 보장
- 회피 신호 감지 시 조기 완성으로 사용자 피로 차단
- 보완 대화는 최대 2턴
